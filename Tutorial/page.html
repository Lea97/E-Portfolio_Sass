Sass (Syntactically Awsome Stylesheet)

Sass is a Preprocessor of CSS. It offers further functions and possibilities as normal CSS.
This results in some advantages that speak for Sass.


Advantages and Disadvantages

Advantages Disadvantages
Easier maintenance of large stylesheets Heavy debugging
Simplified & extended CSS Further source of error
Saving of time Time to learn Sass
Reduce self-written code

Sass Features

The following is a list of the most important functions that Sass offers.

Feature Explanation
Nested rules Selectors can be nested.
Variablen Constant values can be set centrally.
Mixins Reference a larger part of code so that it can be easily reused elsewhere in the stylesheet.
